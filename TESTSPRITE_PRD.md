# CinePurr - Product Requirements Document

## 1. Product Overview
CinePurr is a real-time, synchronized video-watching and chat platform. Users can create public or private streaming rooms to watch videos together and interact via websockets.

## 2. Core Features to Test
- **Authentication:** Users can register and log in to the application.
- **Room Discovery:** The homepage lists active public rooms, trending rooms, and searching capabilities.
- **Watch Rooms:**
  - Synchronized video playback (YouTube and direct MP4).
  - Real-time chat functionality via WebSockets with typing indicators.
  - Video Queue System: Add videos using TMDB search or direct URLs.
  - Queue voting system to determine the next video.
- **Gamification & Socials:**
  - Users earn XP and level up by joining rooms and sending messages.
  - Daily quests, login streaks, and achievements.
  - Global Leaderboards for watch time, level, and messages sent.
  - Friend requests and user profiles.
